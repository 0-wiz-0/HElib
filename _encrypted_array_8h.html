Free functions for various flavors of cached matrix multiplication.

To save time, the constants that need to be computed during a matrix-vector multiply can be precomputed and stored. One can store these either as polynomials (ZZX's) using a CachedPtxtMatrix, or as DoubleCRT's using a CachedDCRTMatrix. These caches are computed using EncryptedArray methods. Functions corresponding to EncryptedArray::mat_mul_dense Caches are computed using EncryptedArray::compMat_dense These functions are for multiplying dense matrices over Z_{p^r}[X]/(G).
